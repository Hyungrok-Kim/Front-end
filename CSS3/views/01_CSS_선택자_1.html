CSS 선택자 1

선택자란?
특정한 HTML 태그 요소를 선택할 때 사용하는 표기 방식
원하는 태그를 '선택'하여 원하는 스타일로 만들 수 있다.

HTML내의 모든 요소 선택자

HTML 안의 모든 요소(tag)를 선택한다.

태그 선택자

HTML 문서 내에 존재하는 동일한 태그들 모두를 선택하는 방법

        p { 스타일 속성 ... }

클래스 선택자

HTML에서는 특정 그룹으로 여러 태그를 묶을 때 'Class' 속성을 사용한다.
스타일에서는 이러한 클래스 속성들만 선택할 수 있는 선택자도 제공한다. 

 .클래스명 { 스타일 속성 ... }

아이디(ID) 선택자

HTML 문서 내의 특정 요소 하나만을 선택하는 방법

         #아이디명 { 스타일 속성 } 
         아이디를 접근할 때는 #을 이용한다.
