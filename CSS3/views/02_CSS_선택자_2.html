속성 선택자

기본 선택자 뒤에 [ ]를 붙여
특정 속성과 해당 속성의 값을 가지는
태그를 선택하는 선택자 

        선택자[속성=속성값] { . . . } 
        = : 해당 속성값을 가질 때
        ~= : 특정 단어를 속성값으로 가질 때 (띄어쓰기로 구분)
        |= : 특정 단어를 속성값으로 가질 때 ("" 로 구분)
        ^= : 속성값이 특정 단어로 시작할 때 (데이터베이스의 like과 비슷)
        $= : 속성값이 특정 단어로 끝날 때
        *= : 속성값이 특정 단어를 포함할 때

자손 선택자와 후손 선택자

자손 선택자 : 해당하는 선택자 바로 아래에 있는 요소들
후손 선택자 : 하위의 모든 요소 

        자손 : 부모선택자>자손선택자 { . . . }
        후손 : 부모선택자 후손선택자 { . . . }

나는 자손 홍길동입니다.

나도 자손 홍길서입니다.

    나는 자손 리스트입니다.
  • 나는 ul의 자손이자, div의 후손입니다.
  • 나는 ul의 자손이자, div의 후손입니다.
  • 나는 ul의 자손이자, div의 후손입니다.

동위 선택자

동급 관계, 이웃한 요소를 선택하는 선택자

        선택자A+선택자B { . . . } : 바로 뒤의 요소 B를 선택
        선택자A~선택자B { . . . } : ( 1 ~ 10 ) A뒤부터 B까지모든 요소를 선택할 때 사용

반응 선택자

사용자의 움직임(마우스 클릭 등)을 감지하여
해당 동작에 따라 스타일을 적용하는 선택자 

        선택자:active { . . . } : 사용자가 마우스로 클릭했을 때
        선택자:hover { . . . } : 마우스가 해당 요소 위로 올라갔을 때
Active 스타일 테스트 (클릭 효과)
Hover 스타일 테스트 (마우스 접근)

상태 선택자

input 입력 양식에 따라 변화하는 스타일 선택자

        input:checked { . . . } : 체크 상태에 따라 스타일 적용
        input:focus { . . . } : 클릭하여 입력창 활성화 되었을 때 스타일 적용
