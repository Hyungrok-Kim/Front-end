선택자 우선순위

CSS에서는 기본적으로 위에서 아래로 적용을 하기 때문에
같은 태그에 스타일이 여러 번 적용될 경우 가장 마지막에
선언한 스타일로 반영된다.
단, CSS 기준의 적용 우선순위가 있을 경우
해당 우선순위에 따라 CSS가 덮어 씌워진다. 

        < : <
        > : >
        태그 선택자 > 클래스 선택자 > 아이디 선택자
        > 인라인 선택자 > !important
        순으로 우선순위를 가진다.
        인라인 선택자는 직접 태그에 속성을 기술하는 방법
