글자 관련 스타일

color 속성 : 글자 의 색상을 설정

          선택자 { color : 색상명 | 16진수숫자 | RGB색상 | RGBA 값 | HSL 값 | HSLA 값; }

색상값 참조 사이트 : colorpicker

  • 색상명
  • 16진수
  • RGB 값
  • RGBA 값(A: 투명도)
  • HSL 값
  • HSLA 값

text-decoration 속성 : 글자의 밑줄, 취소선 등을 표현하는 속성

        선택자 { text-decoration : none | underline | overline | line-through}
  • 밑줄 표현
  • 취소선 표현
  • 윗줄 표현
  • 하이퍼링크

text-shadow : 글자 그림자 속성

        선택자 { text-shadow : none | 가로거리 세로거리 번짐정도 그림자색상;}
HTML5     HTML5     HTML5     HTML5

white-space : 글자 간 여백 설정 스타일

      선택자 { white-space : normal | nowrap | pre | pre-line; }
