이미지 관련 태그

src 속성으로 이미지의 경로를 설정하여 화면에 이미지를 표시할 수 있다.
절대 경로 :
C:/, D:/ 와 같은 보조 기억장치 시작부터 해당 파일이 위치한 경로를 모두 서술하는 것 ex) D:~/~/~/~/river1.PNG
상대 경로 : 현재 파일의 기준으로 다른 파일의 위치를 찾아가는 것
./ or 바로 파일명으로 시작 : 같은 폴더 안의 위치에서 시작한다.
../ : 현재 파일이 위치한 폴더 바깥으로 나가서 찾는다.
ex)
'0_firstPage.html' : 같은 폴더 안에 있음.
'../index.html' : 현재 폴더를 나가서 찾겠다!
'../../../img.jpg' : 현재폴더에서 나가고 나가고 나가서 찾는다. 총 3번나가서

자주 쓰이는 속성

alt 속성

사진의 경로가 잘못되었거나, 이미지 파일이 훼손되었을 경우,
혹은 화면 낭독기(스크린 리더)를 통해 사진을 표현할 때
대체할 설명 문구를 작성하는 속성이다.

width와 height 속성

사진의 가로, 세로 길이를 설정하는 속성,
화면의 크기에 따라 고정 길이와 가변 길이로 설정할 수 있다.

고정길이 : 화면의 크기가 변해도 사진의 크기는 변하지 않는다.

가변길이 : 화면의 크기에 따라 사진의 크기도 변한다.
