자바 스크립트 개요

스크립트 언어란?
사용자의 요구에 맞게 수행되도록 코드를 간결하게
작성하는 언어, 배우기 쉽고 매우 빠르게 구현이 가능하다.
자바 스크립트
ECMA 스크립트 표준을 따르는 대표적인 웹 기술 구현 언어,
웹 브라우저에서 가장 많이 사용하는 인터프리터언어로
객체 지향 언어이다.
사용 형식
  1. inline 방식 :
    태그 내에 코드를 직접 작성하는 방식(간단한 소스 코드일 경우)
  2. internal 방식 :
    script 태그를 선언하여 내부에
    소스 코드를 방식
  3. external 방식 :
    외부의 .js 파일을 가져와 현재 웹 문서에 등록하는 방식
    (like css link 태그로 가져오는 방식)
작성 시 명심할 사항
소스코드는 함수(≒ 메소드) 단위로 작성하고, 이벤트 속성을 통해 실행한다.
이벤트 속성은 태그마다 지원하는 종류가 다를 수 있으며
일반적으로 on이벤트명 = 실행할 함수명([전달인자 | 매개변수]) 으로
이벤트와 스크립트를 연결(binding) 시킨다.

태그에 직접 스크립트를 작성하여 구동 확인하기(inline 방식)

script 태그를 활용하여 내부에서 작성한 소스코드 실행하기(internal 방식)

별도로 작성한 스크립트 함수 실행하기 (external 방식)
