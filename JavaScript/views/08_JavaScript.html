배열

자바스크립트 배열 :

자바에서는 배열을 선언할 때 특정 자료형 별로 선언이
가능했지만, 자바스크립트는 자료형을 별도로 선언하지 않기 때문에
마치 자바의 list처럼 각종 자료형을 하나의 배열에 담을 수 있다.

배열 선언

new Array() 로 선언할 수도 있고,
[]를 사용하여 선언할 수도 있다.
또한 배열 선언 시 크기를 지정할 수도 있다.

배열 함수

indexOf() : 배열의 특정 요소가 몇번째 방에 있는지 확인해서 해당 숫자를 돌려주는 함수

지역 :   

concat() : 두 개의 배열을 하나로 합치는 함수

join : 배열을 하나의 문자열로 합치는 기능 수행하는 함수

sort() : 배열을 오름차순 정렬하는 함수

shift() : 배열의 맨 앞에 요소를 제거하는 함수
unshift() : 배열의 맨 앞에 요소를 추가하는 함수

slice() : 배열의 특정 부분을 잘라내는 함수
splice() : 배열의 특정 인덱스를 잘라내고
원하는 값을 중간에 추가할 때 사용하는 함수

toString() : 배열을 문자열로 바꿔주는 함수
