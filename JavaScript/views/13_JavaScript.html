자바스크립트 객체

객체 선언 및 호출 방법

자바 스크립트는 객체를 키와 값으로 나누어 선언한다.
키를 속성으로 활용하여 해당하는 값을 식별한다.
또한 선언 시 { } 를 이용하여 선언하며,
속성으로는 모든 자료형이 올 수 있다.

객체의 키 식별하기

객체의 키는 모든 문자열로 작성할 수 있지만,
만약 키 이름에 특수문자가 들어간다면 무조건
[ ] 로 써야 한다.

객체의 메소드(함수) 변수

객체의 속성 중 함수 자료형의 속성을 호출하는 방법

객체와 for-in 반복문

객체 안의 각 속성들은 반복문으로도 접근이 가능하다!
단, 이 때엔 일반 for문이 아니라 for-in문으로 접근해야 한다.

객체에 새로운 필드 추가, 삭제
