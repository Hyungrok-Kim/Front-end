Window 객체

window 객체는 자바스크립트가 가지는 브라우저 최상위 객체이다.
크게 BOM과 DOM으로 나누어 진다.
BOM (Browser Object Model) :
location 객체, navigator 객체, history 객체, screen 객체 . . .
DOM (Document Object Model) :
document 객체, 그 이하의 모든 요소들...
