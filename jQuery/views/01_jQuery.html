제이쿼리 개요

제이 쿼리란?

존 레식에 의해 개발된 JS 경량화(코드가 간결해짐, 속도가 빠름, 구현 생산성이 높음)
스크립트 라이브러리
복잡하고 길었던 자바 스크립트 구문들을 보다 쉽게 사용하도록 만들어졌다.

-- JQuery의 장점 --
1. DOM과 관련된 스크립트를 짧고 쉽게 구현하여 구현속도(생산성)을 향상시킨다.
2. ajax(비동기 통신), 이벤트 처리, 이펙트 등을 폭넓게 지원한다.
3. JQuery와 관련된 확장 플러그인, 오픈 API 를 보다 더, 다양하게 지원한다.

-- JQuery 단점 --
1. JQuery 에 익숙해지면 JS에 약해진다. (난, 더이상 JQuery없이는 살 수 없는 몸이 되었어)
2. 관련 확장 플러그인을 쓰고자 할 때 JQuery 버전이 달라 충돌이 발생할 수도 있다.
3. JQuery 자체도 자바스크립트 안에서 돌아가기 때문에, 순수 자바 스크립트로 직접
구현하는 것보다 속도가 느릴 수 밖에 없다.

제이쿼리 연결하기

1. jQuery 소스 코드를 제공하는 사이트에서
소스코드를 인터넷으로 연결하는 방법 (CDN)
** CDN : Content Delivery Network
2. jQuery 사이트에서 소스 코드를 직접 다운로드 받는 방법

$(document).ready()

현재 웹 문서를 읽은 후(onload) ready() 함수를 실행하여
body.onload 처럼 즉석 실행 함수를 구현할 수 있다.
