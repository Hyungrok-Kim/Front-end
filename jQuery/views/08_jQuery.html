태그 요소 삽입 함수

삽입 함수 Part - 1 -

$(A).appendTo(B) : A를 B요소 안의 마지막 부분에 추가
$(A).prependTo(B) : A를 B요소 안의 첫 부분에 추가
$(A).insertBefore(B) : A를 B요소 앞에 추가 (동위요소)
$(A).insertAfter(B) : A를 B요소 뒤에 추가( . . )

삽입 함수 Part - 2 -

$(A).append(B) : B를 A요소(태그) 안의 마지막 부분에 추가
$(A).prepend(B) : B를 A요소 안의 첫 부분에 추가
$(A).before(B) : B를 A요소 앞에 추가
$(A).after(B) : B를 A요소 뒤에 추가

요소의 복제와 삭제 함수

clone() : 지정한 요소를 복사할 때 사용
remove() / detach() / empty() : 요소를 삭제하거나 안의 내용을 텅 비울 때 사용
